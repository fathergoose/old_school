Sounds like a good idea right?

Specifically Material Design Lite. I got the framework rocking with a nav-bar and standard layout then ran into an issue.

This would need to be repeated on every page

Awe crap. Well, I took care of that and great. I have an about me, an index, an articles, and a link to my gitHub (and another). Wait, I need a link to github on each nav-bar. Awe, crap.
